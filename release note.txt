
pour lancer l'application suivre les etapes suivantes:

0- prerequis à installer:

Docker
Maven
java 8
Build project.
read release note.


1-repertoire initial
  Se deplacer dans le repertoire "principal" ou se trouve le fichier docker-compose.yml

2- Se placer dans le repertoire principal et creer le reseau 
   docker network create networkIdruide

3-Creation des JARs applicatifs
mvn clean install -f catalogservice/pom.xml 
mvn clean install -f orderservice/pom.xml
mvn clean install -f packingservice/pom.xml

3-lancer docker compose dans le repertoire principal
docker-compose up

4-Attendre que tous les services  microservices  et kafka soit totalement

5- ouvrir postman et utilisé ces urls pour les tests ou 
sur browser avec graphiql (http://localhost:10555/graphiql)  

http://localhost:10555/catalog pour  catalogservice
http://localhost:10444/order  pour  orderservice
http://localhost:10333/order pour  packingservice

6-Vous pouvez utiliser les jeu de testsqui se trouve dans le repertoire 
./backend/posmanTests
